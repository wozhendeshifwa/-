创建用户

```bash
adduser myf # 创建用户myf
usermod -aG sudo myf  # 给用户acs分配sudo权限
```

首次首次远程服务器

```bash
ssh root@123.57.180.12
# 第一次连接服务器，会出现提示选项信息，输入yes即可
# 接着，输入密码
# 本地 ~/.ssh/known_host 会多了新服务器信息
```

config配置文件 —— ssh 个性化连接参数配置

```bash
cd ~/.ssh/
vim config
Host aliyun
	HostName 123.57.180.12 # 公网ip
	User root   # 根用户 
	Port 22 # 默认值
```

远程服务器密钥登录

1.ssh 本地生成密钥对

```bash
ssh-keygen # 密钥生成
持续回车
# 本地 ~/.ssh/ 文件夹之下多了两个文件
id_rsa # 私钥  
id_rsa_pub # 公钥
```

2.将公钥部署到远程服务器

```bash
ssh-copy-id  aliyun  # ssh-copy-id git@github.com
# 输入远程服务器密码
# 公钥会自动追加到 远程服务器 ~/.ssh/authorized_keys
```
