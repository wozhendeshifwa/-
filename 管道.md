```bash
HOME
PATH
LD_LIBRARY_PATH：用于指定动态链接库(.so文件)的路径，其内容是以冒号分隔的路径列表。
C_INCLUDE_PATH：C语言的头文件路径，内容是以冒号分隔的路径列表。
CPLUS_INCLUDE_PATH：CPP的头文件路径，内容是以冒号分隔的路径列表。
PYTHONPATH：Python导入包的路径，内容是以冒号分隔的路径列表。
JAVA_HOME：jdk的安装目录。
CLASSPATH：存放Java导入类的路径，内容是以冒号分隔的路径列表。
```



文件管理之外的常用命令

系统状况：

```shell
top # 查看所有进程的信息（Linux的任务管理器）
    # 打开后，输入M：按使用内存排序
    # 打开后，输入P：按使用CPU排序
    # 打开后，输入q：退出
df -h # 查看磁盘使用情况
free -h # 查看内存使用情况
du -sh # 查看当前目录占用的磁盘空间
ps aux # 查看所有进程
w # 展示登录用户
ping www.baidu.com # 网络是否连通
```

文件检索

```bash
find ~/homework/lesson_6/homework_0 -name '*.py' # 搜索某个文件路径下的所有py文件
grep xxx # 从 stdin 中读取如干行数据，如果某行中包含xxx,则输出改行；否则忽略改行
wc -l # 从 stdin 中 读入内容，统计行数
grep xxx # 从stdin中读入若干行数据，如果某行中包含xxx，则输出该行；否则忽略该行。
ag xxx # 搜索当前目录下的所有文件，检索xxx字符串
tree # 文件数结构
echo $PATH | cut -d ':' -f 1-3 # 输出:隔开的1~3内容
echo $PATH | cut -d ':' -f 1,3 # 输出:隔开的第1、第3内容
echo $PATH | cut -c 1,3 # 输出第1、第3个字符
echo $PATH | cut -c 1-3 # 输出1到3字符
sort # 将每行内容按字典序排序
```
